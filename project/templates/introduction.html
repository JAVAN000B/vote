    User Introduction

    This is a game voting website, users should sign up and sign in first, then polls their favorite game, they can create their own favorite game as a option as well! We chose the database Mysql to store date. Steps shown as below:

    1. Create your own account using your email

    2. Sign in, then the VOTE button will be shown on each game card, you can only vote for one game for one ticket

    3. If you don't like the games on the page, you can create your own poll by 'New Vote' button on the top right

    4. All the Statistical data will be shown and refreshed on the main page

    Administrator Introduction

    Administrator can log in as administrator using a special account, admin can delete and edit polls, create a new topics or a new account by the buttons on the right top. Every poll comes from users will be recorded by system, admin can check and delete the records in the console. Steps shown as below:

    1. Using account: admin, password: admin123, ticket the checkbox as well to log in

    2. Once logged in, you will be redirected to the admin console page

    3. First part shows the existed topics name and followed by description, the user who created it, then the tickets, you can edit the game_name, description and tickets

    4. Second part shows the existed users

    5. Third part shows the poll history by users, you can delete one record to deduct the tickets of relevant game
